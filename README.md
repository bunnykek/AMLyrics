# AMLyrics
Fetches plain and synced lyrics from Apple Music

- Get the "auth_bearer" and "media-user-token" from music.apple.com browser cookies.      
- Do the changes in the `config.json` file. 
- `pip install -r requirements.txt`
- `py lyrics.py URL`
